# Python Fast API

Some good examples of using the [Flask framework](https://flask.palletsprojects.com/).


## Instalation

    // your env
    python3 -m venv venv
    . venv/bin/activate

    // in the first time
    pip install fastapi
    pip install uvicorn

    // run
    uvicorn main:app --reload


## Try

+ http://127.0.0.1:8000/
+ http://127.0.0.1:8000/docs
+ http://127.0.0.1:8000/redoc
+ http://127.0.0.1:8000/items/5?q=somequery
